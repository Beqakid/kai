# Carehia Kai PA Mode

Carehia Kai should feel like a personal assistant for both clients and caregivers.

## Phase 1 PA Experience

- person-like Kai avatar
- client and caregiver positioning
- active mic scaffold UI
- no audio recording yet
- guided care-search draft handoff
- approval-gated next steps

## Voice Boundary

The mic scaffold can be visible and clickable, but production voice requires:

- explicit consent before recording
- push-to-talk
- live transcript
- mute and stop controls
- speech-to-text provider
- text-to-speech provider
- interruptible playback
- audit logging

Kai must not record care conversations silently.

## Client PA

Kai helps clients and families:

- explain care needs
- prepare a care-search brief
- understand caregiver comparison
- move toward Carehia-approved next actions

## Caregiver PA

Kai helps caregivers:

- understand how Carehia works
- prepare onboarding information
- explain profile and availability requirements
- understand next steps without changing approvals
