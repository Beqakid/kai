# KAI Phase 1 Boundaries

KAI Phase 1 is guide-only.

Allowed:

- Explain app concepts.
- Guide onboarding.
- Show workflow steps.
- Suggest form content.
- Generate structured website drafts.
- Store session, message, preference, knowledge, and audit metadata.

Not allowed:

- Submit forms automatically.
- Delete data.
- Send emails.
- Change permissions.
- Process payments.
- Deploy code.
- Modify production schemas.
- Publish websites automatically.
- Provide legal, medical, or financial advice as final authority.

Future placeholders exist for voice, interviews, adaptive learning, Durable
Objects, Vectorize, R2, AI Gateway, Queues, and Workflows.
