# Incorporacion a Viliniu

Kai puede guiar al usuario para crear un borrador de sitio web antes del
registro, crear una cuenta para guardar el progreso, configurar el perfil,
agregar contacto y ubicacion, agregar productos o servicios, y preparar la
tienda para revision.

La aprobacion no debe bloquear la configuracion inicial. Debe bloquear solo la
publicacion, la visibilidad en el marketplace, los pedidos, los pagos y el
contacto con clientes hasta que Viliniu apruebe la tienda.
